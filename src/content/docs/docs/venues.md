This page helps organizers add locations and use them on games.

## Before you start

- You must be the competition organizer.
- A saved venue belongs to one league.

## Add a saved venue

1. Open **Manage**.
2. Select your competition.
3. Open **Venues**.
4. Tap **Add venue**.
5. Choose **Search Google Places**, **Drop pin on map**, or **Name only**.
6. Enter the venue details.
7. Save.

<!-- SCREENSHOT: Manage > Venues add venue sheet -->

## Attach a venue to a fixture

1. Create or edit a game.
2. Open the venue picker.
3. Choose **No venue**, a saved league venue, a new venue, or a one-off venue name.
4. Save the game.

<!-- SCREENSHOT: fixture venue picker -->

## Rules & good to know

- Saved venue fields can include name, address, city, capacity, notes, latitude, longitude, and Google Place ID.
- Google Places search requires the app to be configured with a Google Maps API key. If not, use pin or name-only venue.
- Saved venue create, update, and delete actions are organizer-only.
- A saved venue must belong to the same league as the game.
- When a saved venue is attached, Sportykore snapshots the venue name onto the game.
- A one-off venue sends only a venue name. It does not create a saved venue record.
- Deleting a saved venue removes the saved venue link from games, but existing games keep the venue name and lose the map link.
- Public match pages show venue name/address and map/directions when coordinates exist.
